# ai-rep-planner-semantic-search
This project implements an AI-powered agentic planning system combined with semantic search capabilities. The system is designed to interpret user queries, retrieve relevant information based on meaning (not just keywords), and generate structured responses using an intelligent planning approach.

#Key Features
Semantic search using vector-based similarity matching
Agentic planning system for structured decision-making
Context-aware query understanding
Intelligent retrieval of relevant information beyond keyword matching
Modular design for scalable AI pipeline development


#How It Works
User inputs a query
Query is converted into vector embeddings
Semantic similarity search retrieves relevant data
Agent component structures and refines the response
Final output is generated based on context and relevance
